## Clone the project

Clone the project.

- SSH: `$ git clone git@github.com:rladies/blog.git`
- HTTPS: `$ git clone https://github.com/rladies/blog.git`

Open the folder in RStudio.

### Install the dependecies

```r
install.packages(c('blogdown', 'magrittr', 'stringr'))
```
